# MG2

ご覧いただきありがとうございます。  
本作品は、大量のキャラクターを同時に扱うことを目的とした  
ゲームフレームワークおよびそのデモです。

詳しい説明や他の作品は[こちら](https://ngaimanchun.github.io/portfolio/)をご覧ください。

## ファイル

実行ファイル： bin\\MG2Game.exe

Visual Studioソリューションファイル： source\\MG2Game\\MG2Game.sln

## 操作方法

* WASDキー：移動
* 方向キー：カメラ操作
* Jキー：攻撃
* 1～3キー：武器切り替え
